quickFilters Change Log - Version 6.0 Changes
  • Thunderbird 115 switched to the Unified toolbar which generally takes up less space than the separate mail toolbar of previous versions. quickFilters uses a new action button which includes all its commands in a single dropdown menu:

    By default, Add-on buttons always appear on the right of the toolbar, but you can move them to a better position by right-clicking the toolbar and selecting "customize...":

  • News / license warnings are integrated into the new Action button. Highlight colors and placement of important commands on the top spot will guide the user to click the currently most important action:

    When a new version is installed, read the "whats new" splash screen and reset the button:

    When the license is close to expiry, to check the license status:

  • New option to disable notification when a filter is run manually (quickFilters Pro users only).
  • Added traditional Chinese translations for User Interface.
  • For more detail on what functions had to be fixed / rewritten, please read on the Github Issue covering conversion to Thunderbird 115.

Bug Fixes
  • This version includes all bugfixes up to version 5.9.3. Going forward, the code for Thunderbird 115 and for previous versions is in separate branches in order to simplify maintenance and the code base getting polluted by too manually incompatible choices.
