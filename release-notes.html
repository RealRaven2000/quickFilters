quickFilters Change Log - Version 6.1 New functions
  • Implemented command "Create Filter from message" in the context menu in the messages list. Just right-click any message after it was moved to a folder (or tagged if you want to create tags through filters)
  • For more detail on what functions had to be fixed / rewritten for Thunderbird 115, please read on the Github Issue covering conversion to Thunderbird 115.
Bug Fixes
  • The file format of filter backups was broken in 6.0, this function is now restored. [issue 197]
Open Issues / Regressions for Thunderbird 115
  • Eventually, all settings will be displayed in a Thunderbird tab, like the settings of most mail extensions. For this, it is still required to rewrite the dialog in HTML format.
  • Should you encounter any regressions / unusual behavior after updating to Thunderbird 115 please tell us on Github at [issue 181]
