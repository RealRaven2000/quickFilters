  • [issue 12] Pro Feature: Add keyboard shortcuts for Run Filters buttons. These can be enabled and configured on the quickFilters Pro tab. This feature is useful for archiving or anti-Junkfilters which should be configured to only run manually. Default shortcuts are: Shift + F: Run Filters on folder Shift + R: Run Filters on selected mails
